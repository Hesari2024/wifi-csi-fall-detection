# MM-Fi

## Source

MM-Fi is a public multi-modal WiFi sensing dataset. It is referenced in RuView documentation as a supported dataset.

> **Source verification required.** Please refer to the official MM-Fi dataset page or associated paper for authoritative metadata, subject count, license, and download instructions.

---

## Why This Dataset Is Included

MM-Fi is included as a related WiFi CSI sensing dataset entry in this repository's catalog because:

- RuView (a third-party WiFi sensing project referenced in this repository) mentions MM-Fi support in its documentation.
- MM-Fi is relevant to the research area of WiFi CSI sensing, human activity sensing, and pose estimation.
- It may be valuable for future real-data benchmarking and robustness evaluation work.

---

## Relationship to This Repository

- MM-Fi is cataloged as a **related dataset** for future planning purposes.
- This repository has **not yet independently integrated, downloaded, or evaluated** MM-Fi.
- MM-Fi is **not claimed as a fall-detection validation dataset** for this repository.
- Inclusion does **not** imply clinical validation or endorsement of MM-Fi's performance claims.
