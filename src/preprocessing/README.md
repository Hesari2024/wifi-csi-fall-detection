# src/preprocessing

This module will contain Python scripts for **CSI signal preprocessing**, including filtering, normalization, and phase/amplitude processing.

## Planned Contents

- Bandpass and low-pass filtering for CSI amplitude signals
- Phase unwrapping and linear phase sanitization
- Amplitude normalization and standardization
- Windowing and segmentation utilities
- Noise reduction and outlier handling

## Current Status

The current preprocessing utilities are located at `src/preprocessing.py` (flat module). This submodule directory is a forward-looking placeholder.

> Migration will be done carefully to avoid breaking existing imports.
